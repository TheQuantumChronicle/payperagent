PayPerAgent Documentation

PayPerAgent is a blockchain-powered API gateway that enables pay-per-use access to 29 APIs using microtransactions on SKALE Network.

Quick Start

1. Base URL

http://localhost:3000

2. Authentication

Include payment proof in the X-PAYMENT header:

X-PAYMENT: <payment_proof>

3. Example Request

curl -H "X-PAYMENT: test" \
  http://localhost:3000/api/coingecko?per_page=10

Available APIs

GET /api/coingecko

Cryptocurrency market data from CoinGecko

Price: 0.001 USDC

GET /api/ruby/price

Token prices from Ruby.Exchange on SKALE Europa Hub

Price: 0.001 USDC

GET /api/wikipedia/search

Wikipedia article search and content

Price: 0.002 USDC

GET /api/reddit

Reddit posts and discussions

Price: 0.002 USDC

GET /api/github/search

GitHub repository search and user data

Price: 0.003 USDC

Features

Batch Requests

Save up to 30% with volume discounts:

  • 5-9 calls: 10% off
  • 10-19 calls: 20% off
  • 20+ calls: 30% off

Reputation System

Earn loyalty discounts up to 20% based on usage:

  • Bronze: 0 requests
  • Silver: 100 requests, 85% success rate → 5% off
  • Gold: 500 requests, 90% success rate → 10% off
  • Platinum: 2000 requests, 95% success rate → 15% off
  • Diamond: 10000 requests, 98% success rate → 20% off

Multi-Token Support

Pay with USDC, USDT, SKL, or WETH on SKALE Network.

SKALE Network

All transactions settle on SKALE Network with:

  • Zero gas fees
  • Instant finality
  • EVM compatible
